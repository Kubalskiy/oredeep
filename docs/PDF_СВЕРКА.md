# Сверка «Основные механики.pdf» с ORE DEEP

Каждая механика дизайн-дока сопоставлена с текущим состоянием игры. Дворфский
рескин: сумки→сумки находок, девушки→старейшины-артель, протеин→пиво,
провизия→гильдия Рудознатцев, боксёр→дворф.

Легенда статуса: **✅ есть** · **◐ частично** · **⬜ нет** · **↺ по-своему**
(реализовано иначе, чем в доке, но осознанно).

---

## Экипировка и сумки

| Пункт дока | Статус | В игре |
|---|---|---|
| Сумка = гача, 1 сумка = 1 предмет, падает за победу | ✅ | `BALANCE.bags` (perVein 1, perBoss 3), `openBag()` |
| Надеть/Продать, продажа → софт-валюта | ✅ | авто-надевание лучшего, слабое → скупщику |
| Уровни сумки: софт-валюта, время, смена шансов дропа | ◐ | уровень и шансы есть; **апгрейд мгновенный, без таймера** |
| 8 редкостей Common…Cosmic | ✅ | `RAR_NAMES` |
| Auto Roll: 1 сумка / 2 сек, ниже порога — продажа, выше — в очередь | ✅ | `autoOpenBag`, порог `autoRollTier`; на разгоне разбирает всю очередь |

**Доработать:** апгрейд сумки с таймером (как тренировки/скиллы в доке).

## Питомцы

| Пункт дока | Статус | В игре |
|---|---|---|
| Гача питомцев, крутка за ресурс | ↺ | крутится **за золото**, а не за яйца |
| **Яйцо** как ресурс (бой/дейлик/спецбои/магазин/оффер) | ⬜ | нет ресурса яиц |
| Pity: выше крутка — выше шанс редкого | ✅ | `geoWeights(petRolls)` |
| Merge: 3 одинаковых → 1 случайный следующей редкости, на макс. заблокирован | ✅ | `mergePet()`, случайный тип |
| **Crafting** после 3 Legendary → Exotic+ за конкретных питомцев + гемы | ⬜ | нет |
| DoT: Bleed/Shock/Splash, passive ATK семьи | ⬜ | питомец даёт только % к стату |

**Доработать:** яйца как ресурс, крафт питомцев, DoT-эффекты.

## Девушки → Старейшины (артель)

| Пункт дока | Статус | В игре |
|---|---|---|
| Гача за **сердца** | ↺ | крутится за золото, ресурса сердец нет |
| Копия стакается и авто-повышает уровень | ◐ | дубликаты идут в артель, слияние **вручную**, не авто |
| Merge: поглощение материала ≤ своей редкости → +уровень | ✅ | `mergeGeo()` |
| Восхождение: легендарная, нужный уровень, hearts + gems | ◐ | `ascendGeo()` за гемы; **без сердец, без Legendary-условия по буквам** |

**Доработать (опц.):** сердца как ресурс, авто-стак копий при ролле.

## Престиж

| Пункт дока | Статус | В игре |
|---|---|---|
| Долгий reset-цикл, сохраняет banked power + множитель тренировок | ✅ | «Глубинный Зов», эхо + множитель |
| **Вход: весь шмот Космический** | ↺ | **вход по Своду Прогона (этап 1000)**, не по редкости шмота |
| Сброс: сумки→1, валюта→старт, шмот уничтожен, стадия→1 | ✅ | `doPrestige()` |
| Сохраняется: скиллы, девушки, питомцы, стикеры, зал | ◐ | скиллы/старейшины/питомцы/коллекция/борода целы; стикеров/зала пока нет |

**Осознанное отличие:** гейт по Своду читается лучше и уже откалиброван (числа
держатся < 1e9). Гейт «весь шмот Cosmic» ломал бы свежий баланс — оставляю Свод.
