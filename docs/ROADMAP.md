# ORE DEEP — продуктовый роадмап до релиза

Точка отсчёта: 7 июля 2026, прототип v0.14 (ядро GDD MVP+V2, пиксель-арт
пайплайн, Capacitor-обёртка, Platform-стабы, 83 автотеста, офлайн-доход).
Недели условные — сдвигаются целиком, порядок фаз фиксирован.

## Фаза 0 — Фундамент ✅ (сделано)

Ядро боя по балансу референса 1:1, сумка/дроп/гача с pity, коллекции
с сет-бонусами, экипировка на персонаже, процедурный арт (руды, шахтёры,
слои), музыка, скорость ×1–×3, экономика с инвариантом доход/сложность,
тест-харнесс, репозиторий, README сборки.

## Фаза 1 — Вертикальный срез (недели 1–2)

Цель: первые 30 минут игры безупречны.

- Онбординг до второго чертога: гарантированные дропы первых жил,
  скриптованный первый камень, стрелки-подсказки (референс: Kolibri —
  дошедший до «второй шахты» не уходит)
- Дейли-стрик с наградами 3/7/14/30 дней
- Шеринг-карточка (canvas): гном в обвесе + находка + глубина + стрик
- Глубина в метрах вместо «этапа», вехи глубины
- SFX ударов/дропа/крита (WebAudio, без файлов)
- Иконка и сплэш, финализация названия, миграции сейвов
- Выход: билд для друзей-тестеров. KPI: средняя сессия ≥ 8 мин

## Фаза 2 — Техническая альфа (недели 3–4)

Цель: нативный билд с реальными интеграциями.

- Android-сборка (Capacitor), подпись, внутренний трек Google Play
- AdMob rewarded (скорость, офлайн ×2), Firebase Analytics вместо
  стабов, LocalNotifications («сундук полон» через 8 ч)
- Клауд-сейв (Play Games Services)
- Плейтест 10–20 человек, правки баланса по воронке событий
- KPI: crash-free ≥ 99.5%, первый замер D1

## Фаза 3 — Контентная полнота (недели 5–7)

Цель: контента на 2+ недели ретеншена.

- Улучшения-таймеры (GDD §11) — возвраты по расписанию + ad-skip
- Экспедиции (престиж): осколки Горы, подсказка при скорости <15% пика
- Выбор «фармить этаж ×1.5 / копать глубже» (Mr. Mine-напряжение)
- Дейлики-квесты с жетонами (GDD §8), Колесо старателя (GDD §7.5)
- IAP: паки кристаллов + no-ads (Play Billing), кристаллы как валюта
  гачи/ускорений
- KPI: D7 на альфе ≥ 10%, среднее число сессий/день ≥ 3

## Фаза 4 — Софт-лонч (недели 8–10)

Цель: сойтись по юнит-экономике до глобалки.

- Гео-тест: Казахстан/Филиппины (дёшево, близкие паттерны)
- ASO: скриншоты с гномом-маскотом, 15-сек видео (пиксельный замах +
  джекпот-находка), A/B иконки
- A/B онбординга по воронке boss_kill/mine_enter
- KPI: D1 ≥ 40%, D7 ≥ 12%, LTV(30) > CPI, ARPDAU baseline

## Фаза 5 — Глобальный релиз и лайв-опс (недели 11+)

- P2-виральность: Общая Жила дня (один сид на всех, шеринг эмодзи-
  сеткой), стрик-борода на карточке, редкие скриншотогенные события
- Ивенты GDD §9 (5 режимов на ключах, геометрические награды)
- V4: PvP-лиги, артель (гильдии), клауд-ивенты
- Сезоны: тематические чертоги (зимний, юбилейный), боевой пропуск —
  после стабилизации метрик
- Ритм лайв-опс: контентный минор каждые 2 недели, ивент каждый месяц

## Риски

| Риск | Митигация |
|---|---|
| CPI выше LTV на софт-лонче | виральные механики P2 раньше графика, крео-итерации |
| Гача-регуляции сторов | шансы опубликованы в игре, pity открыт (уже есть) |
| Один файл index.html разрастается | резка на модули при переходе к Фазе 3 |
| Выгорание контента у топ-игроков | экспедиции + ивенты добавляют глубину без нового арта |

## Критерии go/no-go глобалки

D1 ≥ 40% · D7 ≥ 12% · crash-free ≥ 99.5% · LTV(30) ≥ CPI × 1.2 ·
органика ≥ 15% инсталлов (виральные механики работают)
